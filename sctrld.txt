параметры файла настроек scc.config

  NetflowAdress  - ip и порт на которм слушать netflow. Пример: "0.0.0.0:2055"
  NetflowBufferSize - ахз зачем оно,
  WebPort - Порт на котором работает WEB сервер. Пример: "8080"
  CmdDown - команда при достижении лимита. %s - IP адресю Пример "speed-set %s 0.5"
  CmdUp - команда при увеличении лимита. %s - IP адресю Пример "speed-set %s 100"

запросы к веб серверу

v1/add/
   Увеличить лимит для IP адреса  
параметры:
   ip - ip адрес
   limit - лимит трафика для ip адреса
   offstart - час с которого не считать трафик 
              например 1, значит трафик в 1.00 часов посчитан не будет
   offstop - час до которого не считать тарфик
             например 2, значит трафик в 2.00 часов будет посчитан
пример: 
   http://localhost:8080/v1/set/?ip=127.0.0.1&limit=200&offstart=1&offstop=2

v1/add/
   Увеличить лимит для IP адреса  
параметры:
   ip - ip адрес
пример: 
   http://localhost:8080/v1/add/?ip=127.0.0.1&limit=100

v1/get/
   Получить статистику по IP адресу
параметры:
   ip - ip адрес
пример: 
   http://localhost:8080/v1/get/?ip=127.0.0.1

runtime/
   Получит статистику по приложению. 
пример: 
   http://localhost:8080/runtime/
